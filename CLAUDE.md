# FENRIR — Pump.fun Solana Trading Bot

## Project Overview
Async Python trading bot for sniping memecoin launches on Pump.fun (Solana). Features AI decision engine (Claude via OpenRouter), WebSocket monitoring, direct bonding curve trading, Jito MEV protection, multi-source price feeds, risk management, FastAPI REST API, SQLite persistence, and Rich terminal dashboard.

## Tech Stack
- Python 3.12+ / asyncio / aiohttp / websockets
- Solana SDK (solana, solders, base58)
- FastAPI + Uvicorn (REST API + WebSocket server)
- SQLite (trade database)
- Claude AI via OpenRouter (decision engine)
- Rich (terminal dashboard)
- pytest + pytest-asyncio (testing)

## Key Directories
- `fenrir/` — Main package (bot, config, core, trading, protocol, data, ai, ui)
- `api/` — FastAPI REST backend
- `config/` — default.json, .env.example
- `tools/` — Backtesting framework
- `tests/` — Test suite
